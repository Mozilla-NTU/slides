HTML5 is the combination of newer, standardized markup and APIs.
http://upload.wikimedia.org/wikipedia/commons/f/f7/HTML5-APIs-and-related-technologies-by-Sergey-Mavrody.png

Semantic Web

Semantic tags allow for users and search engines to find, index, share, and combine info easier.

New APIs

New APIs that allow for vector based drawings, 2D and 3D graphics, manipulating, synthesizing, and capturing audio and video, persisting data, safe multithreading, real time communication, message based transport protocols, and many more.

Competition

Browser Engines & JavaScript VMs
  • Gecko - SpiderMonkey
  • Blick - V8
  • WebKit - JavaScriptCore
  • Trident - Chakra

Developers, and thus users, win.

Standards Based

Four major browser engines with similar implementations. Many organizations are a part of the W3C, WHATWG, Kronos, IETF. Not controlled by any one Company.

Universal

Most major consumer electronic devices now feature some form of a web browser.

Write Once, Run Everywhere

Lofty goal, not always achieved, but the best platform to target multiple audiences and multiple devices.

Compile to JavaScript

Many languages can be converted to JavaScript.

Library Support

There are tons of free libraries to help you get started. Such libraries provide powerful abstractions for APIs like DOM manipulation and animation, feature detection, MVC, WebGL, game engines, etc.
