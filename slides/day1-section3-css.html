CSS

Cascading Style Sheets

CSS is a style sheet language used for describing the presentation semantics of a document written in a markup language.

Syntax

Selectors

  • Elemento: h1 {....}
  • Class: .class {....}
  • Id: #class {....}
  • Attribute: input[type='radio'] {....}
  • Universal: * {....}
  • Other: new from CSS3
