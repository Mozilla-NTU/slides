HTML5

HTML5 is the combination of newer, standardized markup and APIs.

Semantic Web

Semantic tags such as <article>, <footer>, etc. allow for users to find, share, and combine information easier.

New APIs

New APIs that allow for vector based drawings, 2D and 3D graphics, manipulating, synthesizing, and capturing audio and video, persisting data, safe multithreading, real time communication, message based transport protocols, and many more.

Competition

Browser vendors are racing to add new APIs or improve the performance of their JavaScript virtual machines to differentiate thier browsers. Developers, and thus users, win.

Standards Based

Four major browser engines with similar implementations. Many organizations a part of the W3C, WHATWG, Kronos, IETF. Not controlled by any one Company.

Universal

Most major electronics now feature some form of a web browser.

Write Once, Run Everywhere

Lofty goal, not always achieved, but the best platform to target multiple audiences.

Compile to JavaScript

Many languages can be converted to JavaScript.
